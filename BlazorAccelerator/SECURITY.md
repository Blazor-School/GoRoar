# Security Policy – Blazor Accelerator

Blazor Accelerator takes security issues seriously.

If you discover a security vulnerability, **do NOT open a public issue.**

Please report all security vulnerabilities privately by email: info@blazorschool.com

Include:

- A clear description of the issue.
- Steps to reproduce.
- A proof of concept if available.
- The affected versions.

We will acknowledge your report within 72 hours and work with you to resolve the issue.

Public disclosure of security issues without coordination is prohibited.
